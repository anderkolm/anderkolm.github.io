// sobre

Sistemas que falham,
oscilam, emergem.

Graduando em Engenharia Mecânica na UFRJ. O fio condutor: entender como estruturas industriais — máquinas, redes de processo, pipelines — se comportam como sistemas dinâmicos não-lineares.

Usar esse entendimento para construir diagnósticos, controladores e monitoramentos mais robustos. Portfolio como experimento: repositórios concisos, código + gráficos, setups reproduzíveis. Preferência por embarcar experimentos a escrever documentação longa.
